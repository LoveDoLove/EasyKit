Installation & Getting Started

System requirements

  • Windows 10/11, macOS 10.15+, or a modern Linux distribution
  • .NET runtime for packaged desktop builds (check release notes)
  • Recommended: Git. Node.js (for NPM tools). PHP & Composer (for Laravel tools).

Install

Choose your platform and download the latest release from GitHub Releases: Releases.

Windows (installer)

  1. Download EasyKit-win-x64.exe from Releases.
  2. Run the installer and follow the prompts.
  3. Optionally enable context menu integration when prompted.

macOS (dmg)

# Mount the DMG and copy the app to /Applications
hdiutil attach EasyKit-macos-x64.dmg
cp -R /Volumes/EasyKit/EasyKit.app /Applications/
hdiutil detach /Volumes/EasyKit

Linux (tar)

# Extract and move to /usr/local/bin for system-wide use
tar -xzf EasyKit-linux-x64.tar.gz
sudo mv EasyKit /usr/local/bin/

Quickstart — first run

  1. Launch EasyKit from Start / Applications or run EasyKit in a terminal.
  2. Open a repository or choose "Open Folder" to point at your project.
  3. Explore the main menu: Git, NPM, Composer, Laravel, Settings, Tool Marketplace.

Configure tool paths

If a tool is not detected automatically, set its absolute path in Settings → Tool Paths (for example, C:\Program Files\Git\bin\git.exe).
