Contact & Support

For questions, bug reports, and contributions use the project GitHub repository:

  • Repository: github.com/LoveDoLove/EasyKit
  • Issues: Report bugs or feature requests
  • Discussions: Ask usage questions and get community help
  • Releases & downloads: Releases

Contributing

See the repository README for contribution guidelines. Typical workflow: fork → branch → tests → PR. Be sure to include a clear description and reproduction steps for issues.

Security

Do not disclose security vulnerabilities publicly. Create a private security advisory or contact the maintainers through the repository's security contact. Include reproduction steps, affected versions, and logs where possible.
