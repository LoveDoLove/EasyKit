Features

EasyKit consolidates common developer workflows into a single, extensible toolkit. Below are the major features and short usage notes.

Git — Guided repository workflows

Interactive repository management: initialize, clone, stage, commit, branch, merge, and push with guided prompts.

Example: commit and push via CLI integration

# Open a repo and run the guided commit flow
EasyKit --open /path/to/repo
# Choose "Git Tools" → "Commit Changes"

NPM — Package management

Inspect, update, and manage Node packages. Run audits, update dependencies, and execute build scripts from within EasyKit.

Composer & Laravel — PHP ecosystem

Composer dependency management and Laravel-specific helpers (artisan commands, migrations, environment helpers).

Tool Marketplace

An extensible marketplace to discover, install, and publish community tools. Tools provide manifests that register commands, UI entries, and configuration schemas.

Automation & Workflows

Combine common tasks into repeatable workflows (example: build → test → package). Workflows can be saved and reused across projects.

Extensibility & Configuration

Plugin manifest model supports commands, pre/post hooks, and UI entries. Configuration profiles allow switching between development and CI settings.
