About EasyKit

EasyKit is a modular, menu-driven Windows toolkit for web developers. It streamlines common development and maintenance tasks by providing a unified console UI for Git, NPM, Composer, Laravel, and more.

With built-in tool detection, context menu integration, and a marketplace for essential developer tools, EasyKit helps automate workflows and reduce repetitive work. Built with .NET 8.0+ and designed for Windows 10/11.

Getting Started

Get EasyKit up and running in minutes

1
Prerequisites

Ensure you have Windows 10/11 and .NET 8.0 SDK installed on your system.

2
Clone Repository

Download the source code from GitHub using git clone command.

git clone https://github.com/LoveDoLove/EasyKit.git
3
Build Solution

Open the solution in Visual Studio 2022+ and build the project.

4
Run EasyKit

Launch the application and start automating your development tasks.
